Sub-seasonal to Seasonal Forecasting for Ethiopia using Deep Learning

Hybrid Deep Learning Framework

The ET-NeuralCast architecture is a Hybrid Deep Learning framework specifically designed for Sub-seasonal to Seasonal (S2S) downscaling. Fusing the Generator (U-Net) and the Discriminator (PatchGAN) to capture the spatiotemporal patterns of the input climate data.

Intelligent Data Fusion

Built on a 4D tensor of atmospheric and oceanic predictors: Geopotential (Z500), Humidity (Q), Wind Vectors (U/V), and Sea Surface Temperature (SST), all cross-referenced with 0.05Â° CHIRPS precipitation history.

Daily Production Pipeline

Fully autonomous CI/CD pipelines running every 24 hours at 00:00 UTC. The system automatically orchestrates data acquisition, inference, and multi-perspective forecast generation.

Data Pipeline

Preprocessing & Normalization

Sources: ERA5 reanalysis and CHIRPS precipitation datasets. Performs clipping to Ethiopia's bounding box and weekly mean aggregation.

  • Anomaly Creation: Subtracts training-set-only weekly climatology.
  • Standardization: Zero-mean/unit-variance scaling using training statistics.

Model Architecture

U-Net Generator Core

Deep encoder-decoder with skip connections ensuring preservation of both local and contextual atmospheric features.

  • Encoder: Progressive downsampling.
  • Decoder: High-res reconstruction.
